# Booza Think Commercial Architecture (상업 및 비즈니스 아키텍처)

---

## 1. 상품 패키지 등급 설계 (Commercial Tiers)

플랫폼은 다양한 유료 모델을 포괄하기 위해 다음과 같은 6개 핵심 라이선스 등급을 기본 정의합니다.

- **Free (무료 등급)**: 기본 조회 기능 제공, 데이터 분석 주기 지연, 낮은 Quota 한도
- **Basic (기본 등급)**: 표준 비즈니스 기능 제공, 일일 데이터 갱신, 기본 지원
- **Premium (프리미엄 등급)**: 인공지능 의사결정 추천, 실시간 시뮬레이션 기능 오픈
- **Enterprise (기업 등급)**: 무제한 Quota, Dedicated DB 복제, API 연동용 개발 라이선스 제공
- **Church (교회 전용 등급)**: 계정과목 매핑, 자동 결재선 설정, 정기 감사 보고서 생성
- **Education (교육 전용 등급)**: 학술 연구용 데이터 벌크 분석, 학습용 샌드박스 무료 크레딧 제공

---

## 2. 요금제 청구 모델 (Billing Models)

1. **Subscription Billing (구독 과금)**
   - 매월/매년 고정 사용료를 청구하며 등급별 Quota(예: 월간 전표 생성 수 100건, API 호출 1000회)를 재부여합니다.
2. **Pay-Per-Use Billing (사용량 기반 과금)**
   - 기본 한도를 초과하여 사용하는 리소스(LLM 토큰, OCR 문서 정제, AI 미디어/동영상 생성 시간)에 대해 구간별 단가표를 적용하여 후불 정산합니다.
3. **License Billing (사용자 수/테넌트 라이선스)**
   - 부서 결재선 및 멤버 수에 비례하여 라이선스를 패키징합니다.
